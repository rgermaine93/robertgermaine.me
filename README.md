# Contact Form - Homepage v2

A clean and modern contact form implementation based on the Figma design.

## Features

- Responsive design that works on all screen sizes
- Clean, modern UI matching the Figma specifications
- Form validation (HTML5)
- Accessible form fields with proper labels
- Hover and focus states for better UX

## Layout Specifications

- Container: 1440px width, 1024px height (responsive)
- Padding: 24px vertical, 240px horizontal (responsive)
- Gap: 86px between container and form
- Flex layout with column direction, centered alignment

## Usage

Simply open `index.html` in a web browser to view the contact form.

For development with a local server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.
